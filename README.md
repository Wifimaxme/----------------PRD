
  # Дизайн сайта по PRD

  This is a code bundle for Дизайн сайта по PRD. The original project is available at https://www.figma.com/design/qcrvrz3w8zqss1OuBhlMw1/%D0%94%D0%B8%D0%B7%D0%B0%D0%B9%D0%BD-%D1%81%D0%B0%D0%B9%D1%82%D0%B0-%D0%BF%D0%BE-PRD.

  ## Running the code

  Copy `.env.example` to `.env` and set `GEMINI_API_KEY` if you want the quiz guide and AI nutritionist tools to use Gemini.

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
